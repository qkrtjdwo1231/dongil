# 동일유리 작업관리 - 서비스 아이디어 정리

## 서비스 개요 및 핵심 목적

동일유리 작업관리는 청주 소재 유리/창호 제조·시공 업체의 내부 엑셀 등록 업무를 웹 기반 업무 화면으로 전환하기 위한 직원용 MVP 서비스다.

이 서비스의 핵심 목적은 직원이 엑셀 파일을 직접 수정하거나 관리하지 않아도, 웹 화면에서 주문 및 작업 데이터를 더 빠르고 정확하게 등록할 수 있도록 돕는 것이다.

특히 기존 거래처, 현장, 품목 정보를 다시 검색하고 재활용할 수 있게 하여 반복 입력을 줄이고, 실제 업무 흐름에 맞는 등록 경험을 제공하는 데 집중한다.

## 주요 기능 목록

- 기본 주문 등록
- 빠른 문장 입력
- 3단계 등록 폼
- 빠른 등록
- 직전 주문 불러오기
- 즐겨찾기 저장 및 재사용
- 거래처별 기본값 추천
- 수량 빠른 입력
- 주문 목록 조회 및 상태 관리
- 확인필요 목록 조회
- 역할별 화면 전환 UI

## 타겟 사용자

- 주문 및 작업 데이터를 직접 등록하는 직원
- 추후 진행 현황과 납기 상태를 확인할 팀장
- 추후 전체 주문 현황과 거래처별 통계를 확인할 대표

현재 1차 MVP에서는 직원 화면을 우선 구현하고, 팀장 및 대표 화면은 자리만 먼저 구성한다.
